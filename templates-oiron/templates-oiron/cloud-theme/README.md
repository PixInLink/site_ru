# Oiron Cloud Pro — Template Pack for GitHub CMS

> Adapted from ThemeForest [Oiron — React JS Landing Page Collection](https://themeforest.net/) Cloud Theme.
> A SaaS-optimized design system with sky-blue palette, glass-morphism cards, and Montserrat typography.

## Target Audience

Cloud services, SaaS startups, IT companies, DevOps platforms, API-first products.

## Design System

| Token | Value | Usage |
|---|---|---|
| Primary (accent) | `#03A9F4` | Links, buttons, accents |
| Primary hover | `#01579B` | Button hover, active states |
| Primary light | `#B3E5FC` | Tags, backgrounds |
| Secondary | `#FF9800` | Highlighted buttons, badges |
| Accent dark | `#1A237E` | Text headings, deep indigo |
| Background | `#FAFAFA` | Page background |
| Surface | `#FFFFFF` | Cards, panels |
| Font | Montserrat 400/500/600 | All text |
| Icons | Material Icons + Ionicons | CDN-linked |

## What's Included

- **theme.scss** — Full Bootstrap 5 design system with CSS variables
- **AI Prompts** — Article + Landing page generation prompts
- **GEO Blocks** — 8 structured content blocks (hero, feature-grid, benefits, etc.)
- **Schema Map** — Block-to-Schema.org type mappings
- **Demo Content** — Sample pages
