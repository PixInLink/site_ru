<p>PixInLink поддерживает 6 стилей генерации. Выбор стиля влияет на всё: от восприятия контента до скорости загрузки и конверсии. В этой статье разберём каждый стиль с примерами URL — чтобы вы могли сразу скопировать и вставить.</p>
<!-- @block: answer-first -->
**Выбирайте стиль под задачу, а не под настроение.** `realistic` — для сайтов и блогов. `illustration` — для лендингов и стартапов. `3d` — для продуктов и SaaS. `cyberpunk` — для игр и IT. `anime` — для творческих проектов. `pixel-art` — для ретро-эстетики.
<!-- @block: key-facts -->
## Как добавить стиль в URL
Параметр `style` добавляется в конец URL:
```
https://pixinlink.ru/800x400/промпт?style=realistic
```
Если стиль не указан, используется `realistic` по умолчанию.

## 1. Realistic (по умолчанию)
Фотореалистичный стиль. Идеален для бизнес-сайтов, блогов, новостей и e-commerce. Изображения выглядят как реальные фотографии.
```
https://pixinlink.ru/800x400/офис-разработчиков?style=realistic
```
**Когда использовать:** og:image, карточки товаров, обложки статей, деловые презентации.

## 2. Illustration
Плоский векторный стиль. Чистые линии, минимализм. Идеален для стартапов, SaaS-лендингов и продуктовых сайтов.
```
https://pixinlink.ru/800x400/стартап-команда-работает-над-продуктом?style=illustration
```
**Когда использовать:** лендинги, разделы «Как это работает», презентации продукта.

## 3. 3D
Трёхмерный рендер с глубиной и объёмом. Подходит для продуктовых сайтов, SaaS и технологических проектов.
```
https://pixinlink.ru/800x400/3d-рендер-облачного-сервера?style=3d
```
**Когда использовать:** хедеры сайтов, продуктовые карточки, tech-блоги.

## 4. Cyberpunk
Футуристический стиль с неоном и темными тонами. Для игр, IT-стартапов, киберспорта.
```
https://pixinlink.ru/800x400/ночной-город-неон-дождь?style=cyberpunk
```
**Когда использовать:** игровые сайты, киберспорт, технологические мероприятия.

## 5. Anime
Стиль японской анимации. Мягкие линии, выразительные персонажи. Для творческих проектов.
```
https://pixinlink.ru/800x400/девушка-в-библиотеке-читает-книгу?style=anime
```
**Когда использовать:** творческие блоги, обложки книг, развлекательные проекты.

## 6. Pixel Art
Ретро-пиксельная графика. Ностальгический стиль 8-битной эры.
```
https://pixinlink.ru/800x400/старый-компьютер-на-столе?style=pixel-art
```
**Когда использовать:** инди-игры, ретро-сайты, ностальгический контент.

## Сравнительная таблица

| Стиль | Размер файла | Визуал | Лучше всего для |
|-------|-------------|--------|----------------|
| Realistic | ~60 КБ | Фото | Блоги, e-commerce |
| Illustration | ~40 КБ | Вектор | Лендинги, SaaS |
| 3D | ~70 КБ | Объём | Продукты, хедеры |
| Cyberpunk | ~55 КБ | Неон | Игры, IT |
| Anime | ~50 КБ | Аниме | Творчество |
| Pixel Art | ~25 КБ | Пиксели | Ретро, инди-игры |

## Комбинирование стилей в рамках одного сайта

Рекомендуем придерживаться одного стиля на всём сайте для консистентности. Исключения:
- Блог: realistic для og:image + иллюстрации в тексте.
- E-commerce: realistic для карточек товаров.
- SaaS: illustration для лендинга.
