## Как это работает

1. Вы вставляете URL: `https://pixinlink.ru/800x400/промпт`
2. Сервер генерирует изображение и сохраняет в WebP.
3. CDN (Content Delivery Network) раздаёт изображение из ближайшего к пользователю узла.
4. Заголовок `Cache-Control: public, max-age=31536000, immutable` говорит браузеру кешировать на год.
5. Следующий запрос — мгновенный ответ из кеша браузера или CDN. Без задержки на генерацию.

## Как выбрать формат

PixInLink поддерживает три формата через параметр `format`:
```
?format=webp   — по умолчанию, лучший баланс качество/размер
?format=avif   — максимальное сжатие (на 30% меньше WebP)
?format=png    — для случаев, когда нужно отсутствие артефактов
```

**Когда использовать AVIF:** для сайтов, где каждый килобайт на счету (мобильный-first, высоконагруженные проекты).
**Когда использовать PNG:** для изображений с текстом, графиков, скриншотов интерфейса (артефакты WebP могут искажать мелкий текст).
