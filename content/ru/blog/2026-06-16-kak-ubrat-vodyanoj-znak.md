- **1 домен = 1 правило.** Все URL с привязанного домена отдаются без водяного знака.
- **Без изменения URL.** Те же ссылки, которые работали с водяным знаком — продолжают работать без него.

## Как это работает технически

Когда браузер запрашивает изображение с PixInLink, он отправляет HTTP-заголовок `Referer` с адресом вашего сайта. Сервер PixInLink проверяет этот заголовок: если домен есть в списке привязанных — изображение отдаётся без водяного знака. Если нет — с водяным знаком.

Это означает, что:
- При прямой вставке URL в браузер (без Referer) водяной знак будет.
- При вставке в `<img>` на привязанном сайте — без водяного знака.
- При просмотре через DevTools или curl без Referer — с водяным знаком.
