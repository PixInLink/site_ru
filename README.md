## What is GitHub CMS Pro?

GitHub CMS Pro is the full-featured version of the static site generator. It includes everything from the Free edition plus:

- **30+ premium templates** — professional designs ready to use
- **Full Schema.org** — 22 types for maximum AI visibility
- **Online store** — demo shop with 30 products
- **Payment system** — Tinkoff integration
- **Account portal** — downloads, licenses, templates management
- **Priority support** — response within 24 hours

## Installation (for license holders)

```bash
# 1. Clone with your license key
git clone https://github.com/biznesemarket/githubcms-pro.git my-site
cd my-site

# 2. Install dependencies
npm install

# 3. Configure edition
echo "VITE_EDITION=pro" > .env

# 4. Start development
npm run dev
```
