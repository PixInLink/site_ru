## Статика vs WordPress

| Угроза | WordPress | GitHub CMS |
|--------|----------|------------|
| SQL-инъекции | Высокий | Нет БД |
| Взлом плагинов | 96% атак | 0 плагинов |

## Встроенная защита
CSP в nginx, HSTS, X-Frame-Options, сканирование билда на токены, HTTPS через Let's Encrypt.
