Методология GitHub CMS: 7-этапный процесс создания контента с трёхагентным дебатом (Expert + Marketer + SEO). Результат: статьи 180KB+ с полной Schema.org разметкой.

## 7 этапов

1. **Анализ** — тема, аудитория, поисковое намерение
2. **Исследование** — 20-30 источников, анализ конкурентов, выявление гэпов
3. **Дебат агентов** — Expert (глубина), Marketer (вовлечение), SEO (оптимизация) согласуют структуру
4. **Структурирование** — 24 GEO-блока с приоритетами
5. **Написание** — каждый блок по шаблону
6. **Проверка** — чек-лист: полнота, GEO, SEO, E-E-A-T
7. **Вывод** — YAML Frontmatter + готовый Markdown

## 24 блока
Заголовок → TL;DR → ключевые факты → определение → введение → Featured Snippet → проблема/решение → тех.параметры → сравнение → статистика → HowTo → roadmap → кейсы → FAQ → отзывы → E-E-A-T → источники → CTA.
