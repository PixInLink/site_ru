Markdown — текстовый формат контента, который ускоряет написание на 40% (GitHub, 2025). GitHub CMS использует Markdown + YAML Frontmatter как единственный источник. Преимущества: чистота, версионирование, AI-читаемость.

## Почему Markdown

1. **Скорость** — пишете текст, а не форматируете. Один файл = одна статья.
2. **Git** — полная история изменений. Откат к любой версии за секунду.
3. **Чистый HTML** — 0 скрытых CSS-классов. AI получает структурированный контент.
4. **Портативность** — .md открывается в любом редакторе. Нет привязки к платформе.

## YAML Frontmatter
Блок метаданных в начале файла: title, description, slug, author, schema_type. Напрямую → JSON-LD + мета-теги.

## FAQ

### Сложно ли писать в Markdown?
**Ответ:** Базовый синтаксис за 5 минут: # заголовок, - список, **жирный**, [ссылка](url). Kilo Code автоматизирует Frontmatter.
