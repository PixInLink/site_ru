GitHub CMS включает 45+ промпт-шаблонов. Пользователь заполняет {INPUT} (описание) и {KEYWORDS} (ключевые слова), отправляет в ChatGPT — получает готовый HTML. 6 категорий: главные, статьи, разделы, магазин, о компании, настройка.

## Категории шаблонов

1. **Главная** (5: home-1..5.txt) — от 6 до 21 секции
2. **Статьи** (5: article-1..5.txt) — HOWTO, сравнение, deep dive, дизайнерские
3. **О компании** (11: about-*.txt) — команда, миссия, вакансии, история и др.
4. **Разделы** (3: razdel-1..3.txt) — лендинги разделов
5. **Магазин** (15: shop-*.txt) — витрины, разделы, товары
6. **Настройка** (site-setup.txt) — все конфигурации

## Как использовать

1. Открыть шаблон в VS Code
2. Заменить {INPUT} и {KEYWORDS}
3. Ctrl+A → Ctrl+C → вставить в ChatGPT
4. Скопировать HTML → в .vue файл
5. npm run build → сайт обновлён

## FAQ

### Нужны ли технические знания?
**Ответ:** Нет. Заполняете {INPUT} на русском. ChatGPT генерирует HTML. Технические знания — только для первой установки.
