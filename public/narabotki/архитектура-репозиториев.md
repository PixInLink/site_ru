# Архитектура репозиториев GitHub CMS

**Утверждено:** 2026-05-12

---

## Дерево репозиториев (5 штук)

```
                           РАЗРАБОТКА (все закрытые)
                                    │
                    ┌───────────────┼───────────────┐
                    ▼               ▼               ▼
          hubcms-dot/githubcms  hubcms-dot/    hubcms-dot/
          (общее ядро)          githubcms_free  githubcms_pro
                    │               │               │
                    │          сборка free     сборка pro
                    │               │               │
                    ▼               ▼               ▼
                           ДИСТРИБУЦИЯ
                    ┌───────────────┴───────────────┐
                    ▼                               ▼
          biznesemarket/githubcms        biznesemarket/githubcms-pro
          ОТКРЫТЫЙ (Free, MIT)           ЗАКРЫТЫЙ (Pro)
```

### Описание каждого репозитория

| Репозиторий | Доступ | Назначение |
|-------------|--------|-----------|
| `hubcms-dot/githubcms` | **Закрытый** | Основная разработка. Ядро: Vue, composables, скрипты сборки, JSON-LD, промпт-шаблоны. |
| `hubcms-dot/githubcms_free` | **Закрытый** | Разработка Free-версии. Содержит копию ядра + free-контент. |
| `hubcms-dot/githubcms_pro` | **Закрытый** | Разработка Pro-версии. Содержит копию ядра + Pro-модули (магазин, платежи). |
| `biznesemarket/githubcms` | **Открытый** | Дистрибуция Free-версии (MIT). Пользователи клонируют отсюда. |
| `biznesemarket/githubcms-pro` | **Закрытый** | Дистрибуция Pro-версии. Доступ по лицензии. |

## Поток: ядро → версии

```
Изменение в hubcms-dot/githubcms (push в main)
                    │
        ┌───────────┴───────────┐
        ▼                       ▼
   sync → githubcms_free   sync → githubcms_pro
   (копируется ядро,        (копируется ядро,
    free-контент остаётся)   pro-контент + модули остаются)
        │                       │
   сборка free-ru + free-en    сборка pro-ru + pro-en
        │                       │
        ▼                       ▼
   deploy → biznesemarket/   deploy → biznesemarket/
            githubcms                 githubcms-pro
            (ОТКРЫТЫЙ)               (ЗАКРЫТЫЙ)
```

**Правило:** ядро (`src/`, `scripts/`, `composables/`, `content/templates/`) синхронизируется **из общего репо во free и pro**. Контент и конфигурация у каждой версии **свои**.

## Различия Free и Pro версий

### Free (githubcms.ru / githubcms.com)

**Меню:** Главная | О компании | Услуги | Блог | Контакты
(НЕТ: Раздел 2, Раздел 3, Магазин)

**JSON-LD (минимальный — для прохождения SEO-валидаторов):**
- Organization
- WebSite (с SearchAction)
- Article (на страницах статей)
- BreadcrumbList

**Промпт-шаблоны:** по одному варианту на каждый пункт меню (home-1, about-1, services-1, article-1, contact)

**SEO:** базовые мета-теги (title, description, og:title, og:description, canonical).
БЕЗ: Twitter Cards, расширенных Schema.org типов (FAQPage, HowTo, Product, etc.)

### Pro (githubcms.ru / githubcms.com — платная)

**Меню:** полное (Home, Раздел 1/2/3, Blog, About, Магазин, Contact, Templates)

**JSON-LD:** все 22 типа из `seo-utils.ts` (Organization, WebSite, Article, Person, FAQPage, HowTo, BreadcrumbList, Product, LocalBusiness, AggregateRating, CaseStudy, VideoObject, etc.)

**Промпт-шаблоны:** 45+ шаблонов (article-1..5, home-1..5, razdel-1..3, shop-*, about-*, etc.)

**Pro-модули:** магазин (shop), платежи (Tinkoff), расширенный SEO/GEO-аудит

## Языки: RU и EN

Обе версии (Free и Pro) разрабатываются сразу на двух языках:
- Русский (githubcms.ru)
- Английский (githubcms.com)

Переключение через `VITE_LOCALE` / `VITE_EDITION`. Контент — в `content/{locale}/`.
